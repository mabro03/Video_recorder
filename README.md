# Video Recorder Using OpenCV

A simple video recorder using OpenCV.

## Features
- space bar를 통해 단순 시청과 녹화 모드를 전환할 수 있습니다.
- ESC 키를 통해 화면을 끌 수 있습니다.
- 빨간 원의 존재로 녹화모드인지 단순 시청 모드인지 구분할 수 있습니다.
- 2번을 누르면 FPS가 20, 3번을 누르면 FPS가 30으로 바뀝니다.
